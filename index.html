👤 Add a User إضافة مستخدم

Login: Log in as an Admin.

تسجيل الدخول: سجِّل الدخول كأدمن.

Navigate: Users → Add User.

الانتقال: المستخدمون ← إضافة مستخدم.

Fill Form: Complete all fields.
• Assign Departments, Role & Manager.
• Enter Monthly Salary, Start Date; leave End Date blank if still employed.

املأ النموذج: أكمل جميع الحقول.
• عيّن الأقسام والدور والمدير.
• أدخل الراتب الشهري وتاريخ البدء؛ اترك تاريخ الانتهاء فارغًا إذا كان الموظف لا يزال على رأس العمل.

Create: Click Create. The new user should appear in the Users list.

إنشاء: اضغط إنشاء. سيظهر المستخدم في قائمة المستخدمين.

⭐ Define a Simple Achievement تعريف إنجاز بسيط

Create KPI-based achievements without adding requirements.

أنشئ إنجازات قائمة على مؤشرات الأداء بدون إضافة متطلبات.

Login: Log in as an Admin.

تسجيل الدخول: سجِّل الدخول كأدمن.

Navigate: Achievements → Add New Achievement.

الانتقال: الإنجازات → إضافة إنجاز جديد.

Basic Info: On the English tab, enter Name, Description, and Default Points Per Unit.

المعلومات الأساسية: في تبويب English أدخل الاسم والوصف والنقاط الافتراضية لكل وحدة.

Toggles: Keep Active ON; leave Generic Achievement OFF.

المفاتيح: اترك التفعيل قيد التشغيل واغلق الإنجاز العام.

Departments: Select Associated Departments if needed.

الأقسام: اختر الأقسام المرتبطة إن لزم.

Save: Leave the Requirements area empty and click Save.

حفظ: اترك قسم المتطلبات فارغًا ثم اضغط حفظ.

📊 Define an Achievement with Requirements تعريف إنجاز مع متطلبات

Use weighted rules to auto-calculate achievement points.

استخدم قواعد الأوزان لحساب نقاط الإنجاز تلقائيًا.

Login: Log in as an Admin.

تسجيل الدخول: سجِّل الدخول كأدمن.

Navigate: Achievements → Add New Achievement.

الانتقال: الإنجازات ← إضافة إنجاز جديد.

Fill Basics: Enter Name, Description, Points etc.

املأ الأساسيات: أدخل الاسم، الوصف، النقاط…

Add Requirements: Click Add Requirement, set Conversion Rate & Weight %.

أضف المتطلبات: اضغط إضافة متطلب وحدد معامل التحويل والنسبة المئوية للوزن.

Save: Click Save.

حفظ: اضغط حفظ.

📝 Log an Achievement تسجيل إنجاز

Login: Log in as User, Manager.

تسجيل الدخول: سجِّل الدخول كمستخدم أو مدير.

Navigate: Achievements → Achievement Types Tab.

الانتقال: الإنجازات ← تبويب أنواع الإنجازات.

Select Achievement: Pick desired achievement, enter the Units/Date as applicable.

اختر الإنجاز: حدد الإنجاز المطلوب وأدخل الوحدات/التاريخ حسب الحاجة.

Submit: Click Submit Achievement. Await approval.

إرسال: اضغط إرسال الإنجاز. انتظر الموافقة.

✅ Approve an Achievement اعتماد إنجاز

Login: Log in as Manager.

تسجيل الدخول: سجِّل الدخول كمدير.

Navigate: Achievements → Approvals → Choose a Pending Achievement.

الانتقال: الإنجازات ← الموافقات ← اختر إنجازًا في انتظار الموافقة.

Review: Inspect details, confirm evidence.

مراجعة: تحقق من التفاصيل وأدلّة الإنجاز.

Approve / Reject: Click Approve or Reject.

اعتماد / رفض: اضغط اعتماد أو رفض.
